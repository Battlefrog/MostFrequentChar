# Most Frequent Char

Project for finding the most frequent character in a specified string.

## Example
    ./main
    Please input a string: Hello!
    Most Frequent Character: 'l'
    Frequency of 'l': 2

## Compiling, Running
`make all`, this will create the file `main` in `/bin/`
